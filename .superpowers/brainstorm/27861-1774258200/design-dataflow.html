Key implementation decisions
Drag-and-drop: @dnd-kit
Scale-aware pointer offsets — dnd-kit's useDraggable hook with a custom sensor that divides deltas by the canvas scale so nodes don't overshoot.
Liquid rendering: liquidjs
One Liquid instance shared across nodes. Each NodeDragHandle parses and renders on template/data change via useEffect. Render errors shown inline.
Visual pickers ↔ class string sync
Pickers read from and write to a parsed class token list. Editing the raw string re-parses tokens. A small token map (e.g. text-sm → text-base → text-lg…) drives the dropdowns — unknown classes pass through untouched.
